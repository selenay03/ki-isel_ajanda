#Bu proje, kişisel görevlerinizi takip etmek için hazırlanmış basit bir Python ajanda uygulamasıdır.
Kullanıcı; görev ekleyebilir, silebilir, tamamlayabilir ve tüm görevleri listeleyebilir.

#Menü Açılır

--- Kişisel Ajanda ---
1- Görev Ekle
2- Görev Sil
3- Görev Tamamla
4- Görevleri Listele
5- Çıkış

#Görev Ekle

Seçiminiz: 1
Görev adı: alışveriş yap
Görev eklendi.

#Görev Sil

Seçiminiz: 2
Silinecek görev: alışveriş yap
Görev silindi.


#Görev Tamamla

Seçiminiz: 3
Tamamlanacak görev: alışveriş yap
Görev tamamlandı olarak işaretlendi.

#Görevleri Listele

Seçiminiz: 4
- alışveriş yap --> ✔ Tamamlandı
- ödev yap --> ⏳ Bekliyor

#Çıkış

Seçiminiz: 5
Program kapatıldı.
